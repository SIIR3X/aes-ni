Decrypts a buffer using AES in CFB mode.

CFB decryption mirrors the encryption process and supports arbitrary-length input. The IV must match the one used during encryption.

Parameters
ctxPointer to a valid AES context (initialized with aes_context_init).
iv16-byte initialization vector (IV) used during encryption. Must not be NULL.
inputPointer to the ciphertext buffer.
input_lenLength of the input in bytes.
outputPointer to the buffer that will receive the plaintext. It must be at least input_len bytes long.

void aes_cfb_encrypt ( const aes_context_t * ctx,
const uint8_t iv[16],
const uint8_t * input,

uint8_t * output )

Encrypts a buffer using AES in CFB mode.

Unlike CBC or ECB, CFB does not require padding. It supports input of any length. The IV is used as the starting feedback and must be unique and random.

Parameters
ctxPointer to a valid AES context (initialized with aes_context_init).
iv16-byte initialization vector (IV). Must not be NULL.
inputPointer to the plaintext buffer.
input_lenLength of the input in bytes.
outputPointer to the buffer that will receive the ciphertext. It must be at least input_len bytes long.

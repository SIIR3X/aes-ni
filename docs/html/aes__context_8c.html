Initializes an AES context by expanding the encryption and decryption keys.

Parameters
ctxPointer to the AES context to initialize.
keyRaw AES key (must be 16, 24, or 32 bytes depending on AES version).
key_sizeSize of the key in bytes (must match AES_128, AES_192, or AES_256).
Returns
0 on success, non-zero on failure (e.g., invalid key size or null pointers).

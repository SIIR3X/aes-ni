Encrypts or decrypts a buffer using AES in CTR mode.

Since CTR mode is symmetric, this function is used for both encryption and decryption.

Parameters
ctxPointer to a valid AES context (initialized with aes_context_init).
iv16-byte initialization vector (nonce + counter). Must not be NULL.
inputPointer to the input buffer (plaintext or ciphertext).
input_lenNumber of bytes to process.
outputPointer to the buffer that will receive the output. Must be at least input_len bytes.

Detailed Description

AES block decryption functions (AES-128, AES-192, AES-256) using AES-NI.

This header declares functions for decrypting a single 16-byte block using AES with hardware acceleration via Intel AES-NI intrinsics.

Each function requires precomputed decryption round keys, which can be derived from the encryption round keys using the corresponding inversion functions.

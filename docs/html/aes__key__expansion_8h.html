Detailed Description

AES key expansion and inversion routines (AES-128, AES-192, AES-256).

This header defines functions to generate encryption and decryption round keys for the AES algorithm, using Intel AES-NI intrinsics for efficient performance. These round keys are used during the AES block encryption and decryption process.

Functions are provided for each AES variant:

  • AES-128: 10 rounds, 11 round keys
  • AES-192: 12 rounds, 13 round keys
  • AES-256: 14 rounds, 15 round keys
